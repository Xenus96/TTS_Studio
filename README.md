# SimpleTTSApp version 0.1.0 (CPU-powered)

### This is a simple XTTSv2-powered Text-to-Speech App written in Python, which has the following features: </br>
1. Make a voiceover of a block of text written in one of the following languages: **English, Russian, German, Romanian, French and Spanish.**</br>
2. Allows to **add more text blocks** to render several sentences written in different languages. In this case, the resulting audio file will have all the rendered text blocks **concatenated**.</br>
3. Allows to **import text** from a .txt file with the **"Import Text File"** button.</br>
4. Allows to **select the Speaker** (or the voice type) which will be used to render the resulting audio.</br>
5. Allows to **stop the rendering process** with the **"Stop Render"** button.</br>

**IMPORTANT NOTE**: This is a draft version of the TTS App, which can have bugs and performance issues!</br>

### Required tools:</br>
1. Microsoft C++ Build Tools **(latest)**;</br>
2. Coqui XTTSv2 ML model;</br>
3. Python 3.11 **(not newer)**;</br>
4. the list of other required libraries is shown inside the **"import"** section of the "app.py" file.</br>
